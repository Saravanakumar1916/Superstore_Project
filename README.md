# Superstore Sales Dashboard (Power BI Project)

## Project Overview
This project is a **comprehensive Power BI dashboard** built using the Superstore dataset.  
It provides **sales insights, profit analysis, product performance, and regional trends** to help businesses make data-driven decisions.

## Pages / Dashboard Layout

### **Page 1 – Sales Overview Dashboard**
- **Cards:** Total Sales, Total Profit, Total Orders, Average Discount  
- **Charts:** Bar Chart (Profit by Category), Line Chart (Sales Trend)  
- **Insight Box:** Key insights summary  
- **Slicers:** Category & Region  

### **Page 2 – Sales & Analysis Dashboard**
- **Charts:** 
  - Top 5 Products (Bar Chart)  
  - Bottom 5 Products (Bar Chart)  
  - Category Sales (Donut Chart)  
- **Layout:** BAR 1 & Donut → Middle row, BAR 2 & 3 → Bottom row  

### **Page 3 – Sales Deep Dive Dashboard**
- **Charts:** 
  - Sales Trend (Line Chart)  
  - Top 5 Products (Bar Chart)  
  - Bottom 5 Loss Products (Bar Chart)  
  - Region-wise Profit / Sales (Bar Chart)  
- **Unique:** Page 3 contains a chart not present in Pages 1 & 2  
- **Slicers skipped:** Page 1 slicers filter this page

## Instructions to Use
1. **Download the PBIX file** from this repository  
2. **Open in Power BI Desktop** (version 2022 or above recommended)  
3. Explore all **3 pages** for insights  
4. Use **slicers on Page 1** to filter data across all pages  
5. Charts are interactive; hover to view detailed info
